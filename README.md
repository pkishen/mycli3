# Usage
<!-- usage -->
```sh-session
$ npm install -g mycli3
$ mycli3 COMMAND

$ mycli3 (-v|--version|version)
mycli3/0.1.1 darwin-x64 node-v12.19.0
$ mycli3 --help [COMMAND]
USAGE
  $ mycli3 COMMAND
...
```
